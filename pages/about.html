About Kanji Tester

How to Use the Custom Kanji Quiz

  1. Prepare a list of kanji that you want to test yourself on.
  2. Use an LLM (like ChatGPT or DeepSeek) with this prompt:
    Please take the following kanji, convert them to their hiragana (NOT katakana), and insert them into sentences that are entirely hiragana. Then, create a question and answer list in the following format: [ { question: the sentence in hiragana, answer: the sentence in its normal form (mixed hiragana and katakana) } ] I want the hiragana for the word the question is based off of surrounded by <strong></strong> tags. I want the kanji for the word the answer is based off of surrounded by <strong></strong> tags. I only want one question and answer per kanji. Here are the kanji:
  3. Save the JSON output from the LLM into a file (e.g., my_kanji.json).
  4. Open the Custom Kanji Quiz page and click "Choose File" to select your JSON file.
  5. Write your answers on paper (no input field is provided).
  6. Click "Toggle Answers" to check your results.

Quiz Features

  • Max Question Count: Limit the number of questions shown
  • Randomize Questions: Shuffle the order of questions
  • Invert Questions & Answers: Switch between testing kanji → hiragana or hiragana → kanji
  • Clear File: Remove the current questions and start over
  • Toggle Answers: Show/hide all answers at once
